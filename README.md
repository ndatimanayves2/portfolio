# Ndatimana Yves - Portfolio

A modern, responsive portfolio website built with React.js and Tailwind CSS.

## Tech Stack

- **React.js** - UI framework
- **Tailwind CSS** - Styling
- **Vite** - Build tool

## Setup

```bash
cd frontend-react
npm install
npm run dev
```

Open: `http://localhost:5173`

## Build

```bash
npm run build
```
